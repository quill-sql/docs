## Week of May 11, 2026

### Bug fixes

- **ChartBuilder modal**: Form fields are no longer hidden behind the submit bar in the [ChartBuilder](/react/report-builder) modal. The scrollable form area now extends past the pinned submit row, so the last fields stay reachable at any modal height.
- **Database logos in Next.js apps**: The database selector in [`@quillsql/admin`](/bi-platform/self-host) now renders database logos reliably when used in Next.js apps. Logos are bundled with the package, so they no longer depend on third-party CDNs.
