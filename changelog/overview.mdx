<Update label="May 18, 2026">

### Improvements

**Agent Chat now runs in a sandbox automatically.** When you start a session with the Agent in the admin portal, Quill spins up a sandbox environment for you so the agent can safely make changes without touching production. You no longer need to manually enter or exit a sandbox to try changes.

**Clearer environment selector while in an Agent sandbox.** The environment picker now shows the parent environment name (e.g. `Production`) instead of locking to "Viewing Sandbox," so it's obvious which environment your changes will apply to once promoted. Learn more about [environments](/bi-platform/overview#1-environments).

**Updated sandbox tooltip copy.** The sandbox indicator now reads "Promote your changes to persist them to the production environment," making the promote-to-prod workflow clearer.

</Update>
