---
title: "Chart Editor"
description: "Let your users edit charts with a simple form."
---

```tsx App.tsx
import { QuillProvider, ChartEditor } from "@quillsql/react";

function App() {
	return (
		<QuillProvider publicKey={API_KEY} organizationId={organizationId}>
			<ChartEditor
				isOpen={isOpen}
				setIsOpen={setIsOpen}
				chartId='8fba9b560982'
			/>
		</QuillProvider>
	);
}
```

<RequestExample>
	```jsx Simple Example
	<ChartEditor isOpen={isOpen} setIsOpen={setIsOpen} chartId='263775e2af12' />
	```

    ```jsx With custom components
    <ChartEditor
    	isOpen={isOpen}
    	setIsOpen={setIsOpen}
    	chartId='8c557a938a9f'
    	ButtonComponen={MyButton}
    	SecondaryButtonComponen={MySecondaryButton}
    />
    ```

</RequestExample>

Presents the Quill Chart Builder form as a modal.

<Info>
	Make sure `QuillProvider` is a parent of the `ChartEditor` component.
</Info>

## Props

<ParamField path='isOpen' type='boolean' required>
	Whether the ChartEditor is open
</ParamField>

<ParamField path='setIsOpen' type='boolean' required>

</ParamField>

<ParamField path='chartId' type='string' required>
