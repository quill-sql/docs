---
title: "SQL Editor"
description: "Let your users create their own charts with the power of SQL."
---

```tsx App.tsx
import { QuillProvider, SQLEditor } from "@quillsql/react";

function App() {
	return (
		<QuillProvider organizationId={organizationId} publicKey={API_KEY}>
			<SQLEditor />
		</QuillProvider>
	);
}
```

<RequestExample>
	```jsx Simple Example
	<SQLEditor />
	```

    ```jsx With custom components
    <SQLEditor
    	defaultQuery="SELECT SUM(amount) FROM TRANSACTIONS"
    	ButtonComponen={MyButton}
    	SecondaryButtonComponen={MySecondaryButton}
    />
    ```

</RequestExample>

Allows your users to build and write custom SQL queries and then add those
queries into their dashboard as a metric, chart, or table.

<Info>
	Make sure `QuillProvider` is a parent of the `SQLEditor` component.
</Info>
