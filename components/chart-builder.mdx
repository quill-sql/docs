title: "Chart Builder"

<Note>
	To see an example of a \<ChartBuilder\> component visit the [examples
	page](/components/examples).
</Note>

```tsx App.tsx
import { QuillProvider, ChartBuilder } from "@quillsql/react";

function App() {
	return (
		<QuillProvider organizationId={organizationId} publicKey={API_KEY}>
			<ChartBuilder />
		</QuillProvider>
	);
}
```

<RequestExample>
	```jsx Simple Example
	<ChartBuilder />
	```

    ```jsx With custom components
    <ChartBuilder
    	ButtonComponen={MyButton}
    	SecondaryButtonComponen={MySecondaryButton}
    />
    ```

</RequestExample>

A simple form that lets users turn a SQL query into a chart, metric, or table
and then add that dashboard item into one of their dashboards.

<Info>
	Make sure `QuillProvider` is a parent of the `ChartEditor` component.
</Info>
